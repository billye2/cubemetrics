## Shipped (Session 2)

### Infra / blockers cleared
- **Dev-server 500 on every route — ROOT-CAUSED & FIXED.** The earlier inline-`env()`
  hypothesis was wrong. Real cause: bare `@import "tailwindcss"` makes Tailwind v4
  auto-scan the **whole repo, including `docs/`** — and the handoff notes literally
  quote a broken `bottom-[calc(64px+env(...)+1rem)]` class while describing the bug,
  which Tailwind extracted as a utility candidate and emitted as invalid CSS.
  Self-referential. Fix: `@import "tailwindcss" source("../")` scopes detection to
  `src/`. (`src/app/globals.css`.) See [[cubemetrics-dev-css-500]].
- **Test suite green on Node 26.** Node 26 registers a stub global `localStorage`
  that shadows happy-dom's; pomodoro-view tests hit `undefined.clear()`. Added an
  in-memory `Storage` shim via a vitest setupFile (`tests/setup/dom-localstorage.ts`,
  wired in `vitest.config.ts`). 842→**844** passing.
